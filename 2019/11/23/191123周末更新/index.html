周末电台

前情提要：做了手机桌面，选择了自己的人生法则。感觉混沌因为没看到非常striking的改变，但是能看到一些好东西正在开始。

  1. 感觉嗖的一下一周又过去了。感觉没有做下来什么很有力量的工作。也被迫正视了rw之前因为不细心引致的我们都希望没有发生过的结果。
  2. 我不能说这是百分百的好事，但是我的确感觉到了一些身体、精神上的变化。入睡安慰，起床也慢慢变得可以控制。我匆匆忙忙走过的一天，怎么着也还是做了点东西的吧。
  3. 或许这就够了。这对于我来说就是于昨天相比起来的进步。需要正视的不仅仅是自己的不足，也需要承认自己在当下的旋涡里是不是已经做出了比较ok的选择。anyway, this is me, this is my life.
